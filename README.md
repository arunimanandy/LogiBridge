## Quick local setup

```bash
python -m venv .venv
source .venv/bin/activate
pip install -r requirements.txt
```

Install Mosquitto separately if running MQTT locally:

```bash
sudo apt-get update
sudo apt-get install -y mosquitto mosquitto-clients
mosquitto -v
```

## Suggested execution sequence

1. Generate clean normalization statistics.
2. Generate dataset.
3. Train FP32 model.
4. Convert INT8 model.
5. Run inference service in Docker.
6. Run PSI drift monitoring.
7. Run benchmark and update final report.

```bash
python data_pipeline/preprocessing.py --make-stats --out data_pipeline/training_stats.npy
python training/generate_dataset.py --out training/logiedge_dataset.npz
python training/train_model.py --data training/logiedge_dataset.npz --out training/models
python training/convert_ptq.py --saved_model training/models/fp32_saved_model --data training/logiedge_dataset.npz --out training/models/model_ptq_int8.tflite
python optimisation/benchmark.py --data training/logiedge_dataset.npz --models training/models
```
